# グローバル市場・相関分析マニュアル (Global Macro Logic)

## 1. 米国経済指標の解釈ルール
* **雇用統計 (NFP) / CPI / PCE**:
  - インフレ警戒期: 数字が強い→ 金利上昇 → 株安（特にナスダック）。
  - リセッション警戒期: 数字が弱い→ 景気後退懸念 → 株安。
  - ゴルディロックス（適温）: 「強すぎず弱すぎず」→ 株高。

## 2. アセットクラス間の相関
* **米国債利回り (US10Y) とグロース株**:
  - 逆相関: 10年債利回りが4.0%を超えて急騰すると、ハイテク株（SOX指数）は売られる。
  - 日本株への波及: SOX指数下落は東京エレクトロンやアドバンテストへの売り圧力となる。
* **ドルインデックス (DXY) とコモディティ**: 逆相関。ドル高は金・原油の下落要因。
* **原油 (WTI) と期待インフレ**: 原油高はFRBのタカ派化を招く。

## 3. 恐怖指数 (VIX) の危険水域
* **VIX < 13**: 楽観。突然の暴落に警戒。プットの買い場。
* **VIX 13-20**: 通常運転。押し目買いが機能しやすい。
* **VIX > 30**: パニック。セリングクライマックス接近、絶好の買い場の可能性。

# 中央銀行政策と為替ポジション分析

## 1. BOJ（日本銀行）の深層と市場インパクト

### 1.1 金融政策決定会合
* **イベントリスク**: 年8回開催。結果発表時間（通常11:30〜13:00）はアルゴが最も狂暴化する時間帯。発表が遅れるほど「政策変更の可能性が高い」と市場は解釈し、ボラが急騰する。
* **総裁会見（15:30〜）**: タカ派的発言は即座に「円高・先物売り」のトリガーとなる。

### 1.2 国債買い入れとテーパリングの力学
* 日銀が国債買い入れを減額すると日本の長期金利が上昇。
* 銀行株（TOPIXバリュー）: 利ざや改善の思惑から強力な買いシグナル。
* 不動産株・グロース株: 資金調達コスト増の懸念から売りシグナル。

### 1.3 ETF買い入れの現状
* 日銀は現在、新規のETF買い入れを事実上停止。「前場にTOPIXが-2%下落したら後場に日銀砲が来る」という期待は幻想（Trap）。
* 真のリスク: 日銀が溜め込んだ巨額ETF残高の出口戦略が、中長期的な日本株の上値の重し（Overhang）として機能。

## 2. IMM通貨先物ポジション（シカゴ投機筋の動向）
毎週金曜日（日本時間土曜朝）に発表される、ヘッジファンドやCTAなど「投機筋」の円ポジションの偏りを示す最重要データ。

### 2.1 データラグの補正
* CFTCデータは「火曜日時点」の建玉であり、発表までに3日間のタイムラグがある。
* プロは水〜金のCVDや値動きを加味して「現在の真の建玉」を推測する。

### 2.2 円ショートの極値と「巻き戻し（Short Cover）」の破壊力
* **投機筋の誤解（Retail Trap）**: 「みんな円を売っているから、まだまだ円安になる」と順張りする。
* **プロの真意（Pro Intent）**: 「ポジションが限界まで傾いている。わずかな悪材料が出れば、パニック的な買い戻し（Short Cover）が発生する」と狩りの準備をする。
* **ショートスクイーズ**: 円ショートが一斉に手仕舞いされると、流動性が枯渇し、数円単位の暴れ狂う円高スパイクが発生する。

### 2.3 日経平均への波及メカニズム
1. トリガー: 米CPI下振れ等による米金利低下。
2. 為替の崩壊: IMMの円ショートが一斉に巻き戻され、フラッシュ・クラッシュ級の急激な「円高」が発生。
3. アルゴの連鎖: HFTアルゴが「USD/JPYの下落」を「日経先物の売りシグナル」としてハードコーディングしており、日経225先物に無慈悲な大量の売りが浴びせられる。
4. 結論: 日経暴落を先読みするには「シカゴ投機筋の円ショートの溜まり具合（燃料の量）」を常に監視せよ。

# 市場の季節性とアノマリー大全

## 1. 月初・月末の構造的フロー
* **月末の売り（EOM Rebalancing）**: その月に株価が大きく上昇した場合、月末最終日の大引けにかけて機械的な利益確定売りが発生する。月末の高値づかみは厳禁。
* **月初のアノマリー（初値買い）**: 毎月第1〜第2営業日は新規マネー流入で強い買い圧力が働きやすい。

## 2. 日本市場（JPX）の特有サイクル

### 3月末：期末配当取りとドレッシング買い
* 高配当バリュー株に「配当取り」の駆け込み買いが入りやすい。
* ドレッシング買い: 機関投資家が期末の運用報告書を見栄え良くするため、保有銘柄を買い上げる現象。

### 4月：海外投資家の「新年度買い」
* 海外年金基金などのニューマネーが日本株に配分されやすい歴史的特異月。統計的にも日経平均の勝率が極めて高い月。

### 8月：夏枯れ相場と「お盆ショック」
* 欧米ヘッジファンド勢がサマーバケーションに入り、市場の流動性が極端に低下する。
* 流動性枯渇の「真空地帯」では、仕掛け的な売りに対して買い向かう主体が不在。わずかな悪材料でフラッシュ・クラッシュ級の暴落が発生しやすい。VIX急騰の警戒月。

### 12月：タックス・ロス・ハーベスティング（節税売り）
* 個人の課税年度末に向けて、含み損銘柄が機械的に損切りされる。中小型株（グロース市場）が年末にかけて理不尽に売られる原因。

## 3. 米国市場の特有サイクル

### 自社株買いのブラックアウト期間
* 米国企業は決算発表前の数週間、自社株買いを停止しなければならない。
* 影響: 米国株最大の「買い手」が消滅するため、相場が下落に弱くなる期間。決算シーズン開始前はプットの買いが機能しやすい。

### トリプル・ウィッチング（米国版メジャーSQ）
* スケジュール: 3, 6, 9, 12月の第3金曜日。米国の株価指数先物・指数オプション・個別株オプションの3つが同時満期。
* 影響: オプション・ディーラーによる激しいガンマ・ヘッジが交錯し、論理的な理由なき乱高下が発生する。

## 5. 関税ショック（2025〜2026年）の日本株需給メカニズム

### 直接経路（企業収益の毀損）
- 米国が日本・中国に追加関税を課す → 輸出企業のEPS下方修正 → 外国人株主による日本株売却
- 特に影響が大きいセクター: 自動車（トヨタ・ホンダ）、半導体製造装置（TEL・Lasertec・Advantest）、精密機械

### 間接経路（円高リスク連鎖）
1. 関税 → 米国景気悪化懸念 → 米金利低下
2. 米金利低下 → ドル安・円高
3. 円高 → 日本企業の円換算収益が減少 → 追加の売り
4. HFTアルゴが「USD/JPY下落 = 日経225先物売り」をハードコーディング → 連鎖発動

### 需給上の構造変化
- 関税ショック局面では「海外リアルマネー（年金・政府系）の日本株アンダーウェイト化」が長期化
- CTAは下落トレンドを確認して追随売りを強化（モメンタム強化）
- 国内勢（GPIF・投資信託・事業法人）がカウンターバランスとして機能する構図になりやすい

### 反転・回復のシグナル
1. 米中・米日の関税交渉が妥結または停戦合意のニュース
2. 外国人の現物売り越しが週▼2兆円超になった後に急減（セリングクライマックス通過）
3. 財務省「対内証券投資」統計が買い越しに急反転
4. CTAのポジション反転（先物の海外勢買い越しへの転換）

## 6. 2026年4月以降：関税ショック後局面の需給読み方

### 現局面の構造（2026年4月時点）
2026年4月初旬、トランプ政権による「相互関税」発動を受けて日本株は急落。
週次需給データでは以下の構造が確認されている：

- **海外投資家**: リアルマネー（年金・政府系）が現物を段階的に売却継続。CTAは先物売りで追随。
  一方で、現物の買い越しが観測される週は「リバランス・バイ（下落後の機械的再配分）」の可能性がある。
- **信託銀行（GPIF）**: 急落局面で現物・先物ともに大規模買い越し → 「年金の底値拾い」シナリオが進行中。
  信託銀行の合算買いが+5,000億円超の週は、Zスコアが+2を超えることが多く、過去の暴落底値と重なる。
- **個人投資家**: 典型的な逆張りパターン。海外が売る局面で買い向かっているが、需給の真の転換には海外の反転が必要。
- **自己（証券会社）**: 急落時のデルタヘッジ・ショート増加 → 相場の下落を増幅させる。VIXが30超で自己の先物売りが拡大する局面は過度なリスク。

### この局面でのZスコア活用ルール
- 海外現物のZスコアが**-2.0以下**：売り超過が統計的に異常水準 → セリングクライマックス接近の可能性
- 信託銀行Zスコアが**+2.0以上**：GPIFが歴史的な規模で買い越し → 需給の底打ちシグナル
- 海外先物Zスコアが**-2.0以下 かつ 現物が改善**：CTAの踏み上げ（ショートカバー）接近の可能性

### 回復局面への移行チェックリスト
以下が揃った週が「反転週」の候補：
```
□ 海外現物 Zスコアが -2 以下から急回復（-1以上へ）
□ 海外先物が売り越し→買い越し転換
□ USD/JPY が円安方向に反転（145円方向）
□ 米中・米日の関税交渉で「90日停戦」「除外品目拡大」等のニュース
□ 信託銀行の買いが継続（GPIFが底値の認定を継続中）
```

### 注意：アノマリーとの衝突
- 4月は「海外投資家の新年度買い」アノマリーがある一方、関税ショックで逆方向に引っ張られる。
  両方を勘案し「アノマリー買いが関税リスクに飲み込まれているか否か」を需給データで確認することが重要。
- 4月第1週データで海外が買い越しなら「アノマリー発動の兆し」、売り越し継続なら「関税リスクが優勢」と解釈する。

## 4. SQ（特別清算指数）と「魔の水曜日」
* SQ週の水曜日は、海外ヘッジファンドが期先限月へのロールオーバーや利益確定の反対売買を強行しやすい日。
* Retail Trap: 「月・火と強かったから水曜も上がる」と順張りすると、水曜午後から突然の強烈な反対売買アルゴに巻き込まれる。
