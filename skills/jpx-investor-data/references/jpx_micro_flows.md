# JPX市場参加者の手口分析プロファイル

## 1. 海外勢（CTA/HFT/Macro）
* **Goldman Sachs (GS)**:
  - 特徴: トレンドフォロワーの巨塔。彼らが大きく「Call買い/先物買い」に傾いた時は、強いトレンド発生のシグナルとなることが多い。
  - 注意: SQ週の水曜日に反対売買で手仕舞うクセがある。
* **Morgan Stanley (MUFG)**:
  - 特徴: 長期目線のマクロファンドのフローが入りやすい。

## 2. 海外勢（Market Makers / Arbitrage）
* **ABN Amro (Clearing)**:
  - 特徴: 個人投資家の反対側（カウンターパーティ）となるマーケットメーカー。
  - 解釈: アムロの「Call売り」が多い場合、それは「個人がCallを買っている」ことを意味する。アムロ自体が下落に賭けているわけではない（デルタニュートラル）。
  - 戦略: アムロの建玉が積み上がっている価格帯（Strike）は、強力な壁（レジスタンス/サポート）になりやすい。
* **Societe Generale (SocGen)**:
  - 特徴: 裁定取引（アービトラージ）の主力。オプションと先物のセット注文が多い。

## 3. 国内勢
* **Nomura (野村)**:
  - 特徴: 国内機関投資家や富裕層のフロー。逆張り（押し目買い/戻り売り）傾向がある。
* **SBI / Rakuten / Matsui**:
  - 特徴: 個人投資家（Retail）。彼らのポジションは「養分」になりやすく、彼らが極端に強気の時は天井、弱気の時は底になりやすい。

# 投資主体別売買動向・財務省統計（MoF）分析マニュアル

## 1. JPX投資主体別売買動向（Trading by Type of Investors）
毎週木曜日（第4営業日）の大引け後に発表される、前週一週間（月〜金）の確定データ。

### 主要プレイヤーの解釈ルール
* **海外投資家 (Foreigners)**:
  - 重要度: 最上（日本市場の売買シェア6〜7割）。トレンドセッター。
  - 「両輪買い（Twin Engine）」: 「現物買い」＋「先物買い」。最強の強気シグナル。
  - 「裁定買い（Arbitrage）」: 「現物買い」＋「先物売り」。見かけほど強くない（NTトレードや配当取り等の可能性）。
  - 解釈: 海外勢が2週以上連続で買い越/売り越した場合、トレンドは継続しやすい。
* **信託銀行 (Trust Banks / GPIF)**:
  - 正体: 年金積立金管理運用独立行政法人（GPIF）のリバランス運用が主。
  - 行動原理: 「逆張り」。株価が上昇すると資産配分（ポートフォリオ）の株比率が増えるため、調整売りを出す。株価が暴落すると買いを入れる。
  - シグナル: 株価高値圏での信託銀行の大幅売り越しは、「健全な上昇トレンド」の証左でもあるが、上値の重し（Supply）となる。
* **個人投資家 (Individuals)**:
  - 現金（Cash）: 逆張り傾向。下落時の押し目買いが多い。
  - 信用（Margin）: 「信用買い残」の増加は、将来の売り圧力（シコリ玉）となるため、積み上がりすぎはネガティブ要因。
* **事業法人 (Corporations)**:
  - 正体: 自社株買い。
  - シグナル: 下落局面での強力なサポート要因（下支え）。決算発表シーズン前後はブラックアウト期間で減ることがある。

## 2. 財務省 対外及び対内証券売買契約等の状況 (MoF Stats)
毎週木曜日に発表される、海外と日本の資金流出入データ。

### データの読み方
* **対内株式投資 (Inward Investment in Stocks)**:
  - 意味: 海外投資家が日本株を「買ったか売ったか」。
  - 活用: JPXの「主体別売買動向」の裏付けデータとして使う。
* **対外中長期債投資 (Outward Investment in Long-term Bonds)**:
  - 意味: 日本の投資家（生保・銀行）が海外の国債などを買ったか。
  - 為替への影響: 日本からの「資金流出（円売り/ドル買い）」を意味するため、円安要因となる。

## 3. 分析の鉄則（Strategic Integration）
* **タイムラグの考慮**:
  - これらのデータは「先週」のものである。「今週の月〜水」の直近の先物手口と方向性が一致しているか確認せよ。
  - 例: 「先週は海外勢爆買い（主体別）」だが、「今週月〜水はGSが急激に売り（手口）」の場合、トレンド転換の可能性が高いため、過去データ（主体別）を過信してはならない。

# 機関投資家フローとCVD（累積デルタ）分析マニュアル

## 1. 信託銀行（GPIF等）の「逆張り」メカニズム

### 1.1 基本的行動原理（リバランス）
* **暴落時の行動（買い支え）**: 
  - 株価が急落すると、ポートフォリオ全体の「株式比率」が目標値を下回る。
  - 比率を元に戻すため、下落局面において機械的かつ大規模な「買い」を入れる。
  - 市場への影響: パニック相場における「最後の買い手」となり、強力な下値支持線（サポート）を形成する。
* **急騰時の行動（売り上がり）**: 
  - 株価が急騰すると、ポートフォリオの「株式比率」が目標値を上回る。
  - 利益確定を兼ねて、上昇局面で機械的な「売り」を出す。

### 1.2 チャート上のシグナルと識別
* **特徴的な値動き**: 暴落局面において、特定のキリの良い価格で下ヒゲを伴う反発が繰り返される場合、GPIF系アルゴの「押し目買い」が発動している可能性が高い。

## 2. CVD（累積デルタ）による海外勢フロー分析

### 2.1 CVDが示す「海外勢（CTA/HFT）」の意図
* **CVDの急激な上昇/下落**: 海外ヘッジファンドやCTAによる攻撃的アルゴリズム（ISなど）の稼働を示す。強いトレンドが発生するサイン。

### 2.2 フロー枯渇（CVDフラット化）のサイン
* **現象**: 激しい値動きの直後、出来高が急減しCVDが横ばいになる。
* **意味**: 大口のトレンドセッターが「静観モード」に入った＝真空地帯。方向感のない乱高下が発生しやすい。

## 3. 【実戦的統合】GPIF下支え × CVD枯渇のコンボ解釈

### コンボ発生時の戦略
* **やってはいけない事（Retail Trap）**: すでに大口の売りが枯渇しているのに「突っ込み売り」すること。GPIFの分厚い買い板に養分を提供するだけ。
* **プロの戦略（Pro Intent）**:
  1. 先物トレードの停止: 方向性を当てる順張りトレードは期待値が下がる。
  2. オプション売り戦略への移行: Pinning現象支配下でショート・ストラングルやクレジット・スプレッドを構築し、IVの剥落を狙う。

# 海外投資家の分類と識別ルール

## CTA（トレンドフォロワー型アルゴ）
- **行動原理**: 過去20〜60日のモメンタムに自動反応。「下落が続けば売り」「上昇が続けば買い」の純粋順張り。
- **識別サイン**: 2〜4週連続で同方向の売買が継続する。経済指標や政策ニュースに関係なく機械的に動く。
- **弱点**: トレンド反転時に「踏み（ショートカバー）」や「投げ（ロスカット）」が集中し、逆方向への急加速が起きる。

## マクロ系ヘッジファンド
- **行動原理**: 経済・政策ファンダメンタルズに基づく中期ポジション（数週間〜数ヶ月）。
- **識別サイン**: 経済指標発表、政策決定会合、地政学ニュース直後に方向性が転換する。
- **特徴**: CTAと異なり、逆張り的に「底値での買い」や「高値での売り」をすることもある。

## リアルマネー（年金・政府系ファンド）
- **行動原理**: 長期的な資産配分に基づく買い。値段に鈍感（価格不感応型）。
- **識別サイン**: 下落局面でも現物の大量買いが継続（週+5,000億円超の現物買い越し）。先物をほぼ使わない。
- **出現タイミング**: 急落局面や年度初め（4月・1月）のニューマネー配分期。

## 複合フローの解釈
- 現物売り＋先物売り（両輪売り）: CTAまたはマクロHFが主導。トレンドが確立している。
- 現物売り＋先物買い（裁定的）: アービトラージまたはロールオーバー。方向性ポジションとは限らない。
- 現物買い＋先物売り（逆裁定）: 配当取り・リバランスの可能性。強気シグナルとはならない。

# 自己（証券会社）大規模ポジションの解釈ルール

自己勘定のネットが±5,000億円を超えた場合、以下の優先順位で判断する。

1. **マーケットメーカー在庫（最も多い）**: 顧客の大規模売り注文を受動的にカウンター買いしてインベントリとして保有。顧客売りが解消されれば自己の買いも解消される。需給改善の証拠とはならない。
2. **裁定取引**: 現物と先物の価格歪みを利用した中立的ポジション。方向性なし。
3. **戦略的方向性ポジション（最も稀）**: 自己勘定としての意図的な長期ポジション。

**識別の実践法**: J-NET（立会外）取引比率が高ければ①②の可能性大（クロス取引・ブロック取引）。立会いオンリーで段階的に積み上げていれば③の可能性が増す。

# J-NET取引（Off-Auction）の解釈ルール

* **定義**: 立会外取引。
* **分析ロジック**: 
  - 立会（Auction）の売買とJ-NETの売買を比較せよ。
  - 「立会で+1000枚、J-NETで-1000枚」の場合、「ロールオーバー」または「クロス取引」であり、新規の方向性ポジションではないと判定し重要度を下げること。

# 重要な権利行使価格（Pinning）の特定
* **ルール**: 各週の建玉残高（Open Interest）が最も多い価格帯を「マグネット（磁石）」として特定せよ。SQが近づくと、株価はその価格に吸い寄せられる傾向がある。
