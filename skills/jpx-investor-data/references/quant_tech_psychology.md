# 実戦的テクニカル分析・戦略セットアップ

## 1. インジケーターの優先順位
* **価格 (Price Action)**: 高値・安値の切り上げ/切り下げ。
* **出来高 (Volume)**: 価格変動の裏付け。出来高を伴わないブレイクはダマシの可能性が高い。
* **移動平均線 (MA)**:
  - 200日線: 長期トレンドの分水嶺（グランビルの法則）。
  - 20日/25日線: 短期トレンドの押し目/戻り目。

## 2. 高勝率セットアップ集

### A. ボラティリティ収縮と爆発 (Squeeze)
* 条件: ボリンジャーバンドの幅が極端に狭くなっている（スクイーズ）。
* アクション: バンドが上下どちらかに広がり（エクスパンション）、価格が2σを超えて確定した方向に順張り。

### B. ダイバージェンス (Divergence)
* 条件: 価格は高値を更新しているが、RSIやMACDは高値を切り下げている（逆行現象）。
* シグナル: トレンド転換（天井打ち）の強力な予兆。

### C. フィボナッチ・リトレースメント (Pullback)
* 条件: 強いトレンド発生後の調整。
* エントリー: 38.2% または 61.8% 戻しの水準で、反転のプライスアクション（包み足、ピンバー）が出現したらエントリー。

# 機関投資家の執行アルゴリズムと市場への影響

## 1. ベンチマーク系アルゴリズム
* **VWAP (Volume Weighted Average Price)**:
  - 定義: 売買高加重平均価格。当日の「相場の平均コスト」。
  - 現象: 株価がVWAPから大きく乖離すると、VWAP付近まで戻ろうとする力（Mean Reversion）が働く。
  - 重要性: 機関投資家の絶対的な防衛ライン。
* **TWAP (Time Weighted Average Price)**:
  - 「材料がないのに、一定のリズムで買い/売りが入り続ける」場合、TWAPによるリバランスの可能性が高い。

## 2. 攻撃・ステルス系アルゴリズム
* **Iceberg（アイスバーグ/氷山注文）**:
  - 定義: 巨大な注文を小分けにして板（Order Book）に見せないようにする技術。
  - 検知: 「板にある売り注文を買っても買っても、即座に同じ価格に売り注文が補充される（Refreshされる）」現象（蓋/Lid）。
* **Implementation Shortfall (IS)**:
  - 定義: 「早く執行すること」を優先する急ぎのアルゴリズム。
  - 現象: スリッページを許容してでも成行で叩いてくるため、相場を一方向に急加速させる。

## 3. 個人投資家への示唆
* **「引け値（Close）」の魔術**: 14:30以降（特にラスト15分）にVWAPに合わせるための激しい売買が発生する。デイトレーダーはこの時間のボラティリティに注意。

# クオンツ・統計的市場分析モデル

## 1. Zスコア (Z-Score) と平均回帰
* **公式**: Z = (現在値 - 平均値) / 標準偏差
* **判定基準**:
  - Zスコア > +2.0: 統計的に「買われすぎ」。反落のリスク大。
  - Zスコア < -2.0: 統計的に「売られすぎ」。反発のチャンス（Mean Reversion）。
  - Zスコア > +3.0: 異常事態（Black Swan級）。トレンドが強すぎて逆張りは危険。

## 2. コインテグレーション (NTスプレッド)
* **NTショート/ロング戦略**: 日経平均がTOPIXに対して異常に買われた場合、「日経売り＋TOPIX買い」を行うことで、市場全体の暴落リスクを負わずにサヤ取りができる。

## 3. ボラティリティ・コーン (Volatility Cone)
* **活用**: 「今のIV（インプライド・ボラティリティ）は、歴史的に見て本当に高いのか？」を客観的に判断するために使用する。

# トレーダー心理・メンタル管理プロトコル

## 1. 初心者が陥る「死の7つの罪」
AIはユーザーの言動から以下の兆候を感じたら、直ちに警告せよ。

1. **FOMO (飛びつき)**: 「急騰しているから買わなきゃ！」→ 天井掴みの典型。アドバイス: 「バスはまた来ます。次のセットアップを待ちましょう。」
2. **Revenge Trading (取り返し)**: 損切り直後に倍のロットでエントリーする行為。アドバイス: 「PCを閉じて散歩に行きなさい。感情的なトレードは100%負けます。」
3. **Confirmation Bias (確証バイアス)**: 自分のポジションに有利なニュースしか見なくなる。アドバイス: 「あなたにとって『最悪のシナリオ』は何ですか？それを直視してください。」
4. **Over-leveraging (フルレバ)**: 一発逆転を狙う。
5. **No Plan (無計画)**: エントリー前に「利確」と「損切り」の位置を決めていない。

## 2. プロの資金管理 (Money Management)
* **2%ルール**: 1回のトレードで許容できる損失は、総資金の2%まで。
* **リスクリワードレシオ**: 最低でも 1:2（損失1に対して利益2）が見込める場所でしかエントリーしない。
* **ポジショニング**: 「確信度」が高い時こそロットを張るが、イベント（CPI等）前はロットを落とす。

## 3. 負けた時の処方箋
* ルール通りの損切りなら、それは「経費」であり、生き残ったことを称賛せよ。
* ルールを破った損切りなら、厳しく叱責せよ。
