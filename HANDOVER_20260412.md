## 現在の構成

```
PC起動
  └─ Docker Desktop 自動起動（設定済み）
       └─ n8n コンテナ 自動起動（restart: unless-stopped）
  └─ api_server.py 自動起動（Windowsスタートアップ登録済み）
```

### 毎週の自動実行フロー（木曜20時JST）
```
n8n スケジュール起動
  → Supabase起動確認（HTTPリクエスト）
  → JPX分析実行（POST http://host.docker.internal:8765/run）
       ← api_server.py が main.py を実行
  → 成功判定
  → レポートサマリー取得（GET http://host.docker.internal:8765/summary）
  → 完了メール送信（Gmail SMTP）
```
